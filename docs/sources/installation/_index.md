- [Install on RPM-based Linux (CentOS, Fedora, OpenSuse, RedHat)]({{< relref "rpm" >}})

- [Run Docker image]({{< relref "docker" >}})

For upgrade instructions, refer to [Upgrade Grafarg]({{< relref "upgrading.md" >}}).
